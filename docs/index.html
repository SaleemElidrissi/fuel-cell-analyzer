Performance metric versus temperature.
Method (per Project 2 Additional Details): Erev = welec/(nF), welec = (Σνh̄)react − (Σνh̄)prod − T[(Σνs̄)react − (Σνs̄)prod] = −ΔG. Each species' h̄(T) and s̄°(T) are built from a 298.15 K reference by integrating the ideal-gas heat-capacity polynomial Cp = A+BT+CT²+DT³; air reaction carries N₂; product water is vapour. Fuel phase switches at its boiling point (hexanol 157.1 °C). Efficiency is on a lower-heating-value basis. Same model as the Python source (fuel_cell.py).
